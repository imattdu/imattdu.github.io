





poetry 安装

```shell
curl -sSL https://install.python-poetry.org | python3 -
```

 注意：

- 必须确保你的系统已经有 **Python 3.7+**（推荐 3.11/3.12）。



默认安装在

```
$HOME/.local/bin/poetry
```





```
# poetry
export PATH=$HOME/.local/bin:$PATH
```





```
# 升级 poetry
poetry self update

# 卸载 poetry
poetry self uninstall
```







```
poetry install --no-root
```
