# **✅ 1.** 

# **线条规范（最重要）**







## **1.1 业务调用（App → BFF → Service）**





- **颜色：蓝色 #3498DB**
- **线型：实线**
- **箭头：实心箭头（filled）**
- 用途：同步调用、RPC、HTTP 请求





------





## **1.2 内部服务调用（Service → Service）**





- **颜色：深蓝 #2C3E50**
- **线型：实线**
- **箭头：实心箭头**
- 用途：内部服务调用、平台模块对接





------





## **1.3 数据写入链路（Service → DB/MQ/ES）**





- **颜色：绿色 #27AE60**
- **线型：虚线（dashed）**
- **箭头：空心箭头（open）**
- 用途：数据写入、异步投递、持久化





------





## **1.4 数据消费链路（Kafka → Flink → Service）**





- **颜色：紫色 #9B59B6**
- **线型：虚线（dashed）**
- **箭头：菱形箭头（diamond）**
- 用途：流式消费、事件流、实时数据处理



# **2.** 

# **框（节点）颜色规范**





你系统中的节点可以使用以下颜色体系，专业又清晰：

| **节点类型**          | **示例**     | **推荐颜色**      | **说明**     |
| --------------------- | ------------ | ----------------- | ------------ |
| **终端/客户端**       | App / Web    | #BBDEFB（浅蓝）   | 来源端       |
| **日志采集组件**      | LogAgent     | #B0BEC5（灰灰蓝） | 基础设施     |
| **消息系统（Kafka）** | TopicA/B     | #FFB74D（橙色）   | MQ 组件      |
| **Flink Job**         | 清洗、去重   | #CE93D8（紫色）   | 大数据处理   |
| **业务服务**          | Java/Go 服务 | #A5D6A7（浅绿）   | 在线服务模块 |
| **数据库**            | MySQL、ES    | #FFE082（浅黄）   | 存储层       |
| **外部服务 / 第三方** | 其他平台     | #CFD8DC（浅灰）   | 非核心系统   |
